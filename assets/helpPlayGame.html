The game is started by pressing the Start button at the top of the game page. This starts the clock, which will then count up the minutes and seconds in each half. The clock can be stopped and restarted as many times as you like by pressing the button displaying the time.

Several game management activities can be performed by pressing the device's Menu button. These include:

  1. Undo last event - e.g., to undo a mistaken shot on goal;

  2. Add player - supports adding a new player to the roster during a game;

  3. Edit game - supports changing game level information such as game location, our team name, and their team name;

  4. Help - to access game time help information (i.e., this help info)

Player time can be captured by utilizing Substitutions. Press the Off field/On field button to indicate whether a player is on the bench or playing. On/off field times will be recorded for every player in the roster. Players are sorted first by whether they are on or off the field, then by the amount of time they have been on or off the field, and finally by jersey number.

The 2nd half can be started by:

  1. stopping the clock

  2. pressing the 1st half button to change halves, and

  3. pressing the Start button to start the clock for the second half.

Reverting back to the first half can be accomplished as long as the 2nd half game clock hasn't been started. To undo switching to the 2nd half, press the 2nd half button. The half will be changed back to the 1st half, and the clock can be restarted from where it was last stopped.

Once the game clock is running Goals and Shots on Goal can be recorded. Simply press the Score or SOG button and specify the details as prompted by the app.

Correcting a data entry error, such as mistakenly pressing the Score button, can be accomplished by pressing and holding the associated button until the associated statistic is reset to its previous value. Alternatively, press the Menu button and select Undo last event.

The game can be ended by:

  1. stopping the clock, and

  2. pressing the 2nd half button to end the game.

The 2nd half can be restarted if desired, by pressing the Resume button and restarting the clock. The game statistics are finalized by pressing the End button. At this point the game's summary statistics will be shown. Pressing the device's back button will navigate back to the game and main application screens.
